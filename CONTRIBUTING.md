# Contributing

Thank you for your interest in improving Rodent Manual Scorer.

## Reporting bugs and requesting features

Please open an Issue using the templates provided. For a bug, include your operating system, Python version, the steps to reproduce, and the full error message if there is one.

## Submitting changes

1. Fork the repository and create a branch for your change.
2. Keep changes focused, and test that both `VideoTimer.py` and `VideoTimerWindows.py` still launch and score correctly.
3. Open a pull request describing what you changed and why.

## Code style

- Target Python 3.9 or newer.
- Keep the macOS and Windows scripts behaviourally consistent. The Windows script carries Windows-specific video backend handling, so apply user-facing changes to both.
